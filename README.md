# 🍔 FoodieExpress - Food Delivery Web App

A full-stack Food Delivery Web Application built with the MERN stack (MongoDB, Express.js, React.js, Node.js) featuring Stripe payment integration, user authentication, admin panel, and a responsive UI.

---

## 🚀 Features

### 👨‍🍳 User
- Browse restaurant food items
- Add items to cart
- Secure checkout with Stripe
- View past orders
- User authentication (JWT)

### 🛠️ Admin
- Add/edit/delete food items
- View all orders
- Update order status (e.g., Preparing → Delivered)

### 💳 Payments
- Stripe integration for real-time payments (USD)
- Payment verification and order tracking
